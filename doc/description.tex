\section{Users}

Users should be able to do the following things with regard to other users:

\begin{itemize}
  \item Create
  \item Read
  \item Update
  \item Destroy
\end{itemize}

This should be authorized between \emph{roles}, not on a ``user by user'' basis.
It's a sort of one-way join.
